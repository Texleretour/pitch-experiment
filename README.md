## Getting Started

### Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** (v20 or higher)
- **pnpm** (v8 or higher)
- **Git**

### Installation

1. Clone the repository

```bash
git clone https://github.com/Texleretour/pitch-experiment.
cd pitch-experiment/ 
```

2. Install dependencies

```bash
pnpm install
```

### Running

```bash
pnpm run dev
```
